getao

PURPOSE

GETAO - returns the AcceleratorObject (AO)

SYNOPSIS

function AO = getao(FileName)

DESCRIPTION 

GETAO - returns the AcceleratorObject (AO)
  AO = getao(FileName)

  If FileName = 'Archive', then the AO and AD are saved to the 
                default directory under the data root
  If FileName = a string beside 'Archive', then the AO and AD are saved  
                to that string name

  • channel2dev CHANNEL2DEV - Converts a channel name list to a device list
  • checkforao CHECKFORAO - Run aoinit if no middle layer is present
  • common2channel COMMON2CHANNEL - Convert a common name to a channel name
  • common2dev COMMON2DEV - Converts a common name to a device list
  • getfamilylist GETFAMILYLIST - Returns a list of all the family names
  • isfamily ISFAMILY - True for family names
  • saveao SAVEAO - Saves the AO and AD to a .mat file
  • setfamilydata SETFAMILYDATA - Sets data associated with accelerator control
  • showao SHOWAO - Display the AcceleratorObjects fields that have get or set capability
  • showfamily SHOWFAMILY - Displays a list of all families
