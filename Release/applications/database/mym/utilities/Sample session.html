List functions available for interacting with MySQL

lookfor mysql
 
 MYM - Interact with a MySQL database server 
 DBADD     Create a MySQL database 
 DBCURR    Show current MySQL database
 DBDROP    Delete a MySQL database 
 DBLIST    List available MySQL databases 
 DBOPEN    Open a MySQL database 
 DBRENAME  Rename a MySQL database
 MYCHECK   Show status of MySQL connection 
 MYCLOSE   Disconnect from MySQL
 MYOPEN    Connect to MySQL 
 TBADD     Create a MySQL table 
 TBATTR    Show structure of a MySQL table
 TBDROP    Delete a MySQL table
 TBLIST    List tables in a MySQL database 
 TBPRINT   Show several rows of a MySQL table
 TBREAD    Read from a MySQL table 
 TBRENAME  Rename a MySQL table
 TBSIZE    Show size of a MySQL table 
 TBWRITE   Write to a MySQL table

Presence of MYM.M is required.

Add a new table

Note that input cell arrays can be constructed programmatically, using loops

for i = 1:100
  type{i} = ‘varchar(20)’;
end

or edited in Excel and imported to Matlab

names = xlsread('Book1.xls',1,'A1:A10');
types = xlsread('Book1.xls',1,'B1:B10');

In both cases, cell arrays ouput by a prior TBATTR command can be used.

[old_names,old_types] = tbattr('old_table');
new_names = old_names;
new_names{1} = 'x';
 
tbadd('test_tabble',{'number','name'},{'double(16,2)','varchar(10)'})

Populate the table with TBWRITE

Having to define variables global for read/write operations is a controversial feature of TBREAD and TBWRITE design.

global dayNumbers dayNames
dayNumbers = 1:7;
dayNames   = {'Monday','Tuesday','Wednesday','Thursday','Friday','Saturday','Sunday'};
tbwrite('test_table',{'dayNumbers','dayNames'},{'number','name'})
